A plugin for modeling sites.

== Description ==

Perfect for modeling agencies, this plugin allows you to add models to your sites and attach a featured photo, photos, and videos of the models. Be sure to check out our [pro plugin](http://actionhook.com/downloads/wp-models-pro) which adds premium support and model shoots!

== Installation ==

1. Upload `wp-models.zip` to the `/wp-content/plugins/` directory and extract.
1. Activate the plugin through the 'Plugins' menu in WordPress
